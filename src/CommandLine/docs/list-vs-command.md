
# `list-vs` Command

Lists Visual Studio installations.

## Synopsis

```
roslynator list-vs
[--file-log]
[--file-log-verbosity]
[-v|--verbosity]
```

### Optional Options

**`-v|--verbosity`** `{q[uiet]|m[inimal]|n[ormal]|d[etailed]|diag[nostic]}`

Defines the amount of information to display in the log.
