# RCS0032: Add newline before expression\-body arrow instead of after it \(or vice versa\)

| Property | Value      |
| -------- | ---------- |

| Category | Formatting |
| Severity | None       |

## Example

### Code with Diagnostic

```csharp

```

