# RCS1211: Remove unnecessary 'else'

| Property | Value      |
| -------- | ---------- |
| Id       | RCS1211    |
| Category | Redundancy |
| Severity | Hidden     |

## Example

### Code with Diagnostic

```csharp
if (condition)
{
    return WhenTrue();
}
else // RCS1211
{
    return WhenFalse();
}
```

### Code with Fix

```csharp
if (condition)
{
    return WhenTrue();
}

return WhenFalse();
```
