# RCS0031: Add newline before enum member

| Property | Value      |
| -------- | ---------- |
| Id       | RCS0031    |
| Category | Formatting |
| Severity | None       |

## Example

### Code with Diagnostic

```csharp
enum Foo { A, B, C, }
```

### Code with Fix

```csharp
enum Foo
{
    A,
    B,
    C,
}
```
