# RCS1234: Duplicate enum value

| Property | Value       |
| -------- | ----------- |
| Id       | RCS1234     |
| Category | Readability |
| Severity | Info        |

## Example

### Code with Diagnostic

```csharp
enum E
{
    A = 1,
    B = 2,
    C = 2, RCS1234
}
```

### Code with Fix

```csharp
enum E
{
    A = 1,
    B = 2,
    C = B,
}
```
