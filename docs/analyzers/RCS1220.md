# RCS1220: Use pattern matching instead of combination of 'is' operator and cast operator

| Property                 | Value   |
| ------------------------ | ------- |
| Id                       | RCS1220 |
| Category                 | Usage   |
| Severity                 | Info    |
| Minimal Language Version | 7\.0    |

## Example

### Code with Diagnostic

```csharp
if (x is T && Foo((T)x)) // RCS1220
{
}
```

### Code with Fix

```csharp
if (x is T y && Foo(y))
{
}
```
