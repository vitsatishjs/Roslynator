## Add tag to documentation comment

| Property           | Value                                       |
| ------------------ | ------------------------------------------- |
| Id                 | RR0208                                      |
| Title              | Add tag to documentation comment            |
| Syntax             | selected word\(s\) in documentation comment |
| Enabled by Default | &#x2713;                                    |

### Usage

#### Before

```csharp
/// <summary>
/// null
/// </summary>
public class Foo
{
}
```

#### After

```csharp
/// <summary>
/// <c>null</c>
/// </summary>
public class Foo
{
}
```
